# deepshukla08.github.io

My personal portfolio website — built with Next.js 16, TypeScript, and Tailwind CSS.

## Live

[https://deepshukla08.github.io](https://deepshukla08.github.io)

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Deployment:** GitHub Pages (static export)

## Development

```bash
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build
```

The site is statically exported and deployed via GitHub Pages.
